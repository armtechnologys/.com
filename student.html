Course Information

Day 1: Introduction to Python

  • Introduction to Python and its features
  • Installing Python and setting up the development environment
  • Basic syntax and data types (integers, floats, strings, booleans)

    • Practice programs:

    • Print "Hello, World!"
    • Perform basic arithmetic operations
    • Concatenate strings
    • Check type of different variables
    • Get user input and display it

Day 2: Variables and Data Types

  • Variables and naming conventions
  • Basic data types (int, float, str, bool)
  • Type conversion (int(), float(), str(), bool())

    • Practice programs:

    • Perform arithmetic operations on variables
    • Concatenate strings and convert to different data types
    • Use boolean variables and logical operators
    • Convert temperature from Celsius to Fahrenheit
    • Calculate the area of a circle using radius input from the user

Day 3: Control Flow - Conditional Statements

  • If statements and indentation
  • Comparison operators (>, <, ==, !=, etc.)
  • Logical operators (and, or, not)

    • Practice programs:

    • Check if a number is positive, negative, or zero
    • Determine the largest among three numbers
    • Check if a year is a leap year
    • Determine pass/fail based on exam score
    • Convert numeric grade to letter grade
